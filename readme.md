# 🌍 AI Travel Agent Chatbot (Python + Streamlit)

An AI-powered **Travel Assistant Chatbot** built with **Python**, **Streamlit**, and **OpenAI API**.
It helps users explore destinations, find travel suggestions, and get personalized trip plans — all through natural conversation.

---

## ✈️ Features

* 💬 Interactive chat interface for travel-related queries
* 🗺️ Suggests destinations, itineraries, and travel tips
* 🧠 Built on OpenAI’s GPT model for realistic conversation
* 🌐 Clean Streamlit UI for web deployment
* 🔑 Secure API key management using `.env`
* ☁️ Deployable on Streamlit Cloud, Render, or Hugging Face Spaces

---

## 🧠 Tech Stack

* **Python 3.10+**
* **Streamlit**
* **OpenAI API**
* **dotenv**

## ⚙️ Setup Instructions

### 1️⃣ Clone this Repository


### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate    # macOS/Linux
venv\Scripts\activate       # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Add Your API Key

Create a `.env` file in the project root:

```
OPENAI_API_KEY=your_api_key_here
```

### 5️⃣ Run the App

```bash
streamlit run app.py
```
